# GitHub Discussions Setup (Phase 4)

RCO uses GitHub Discussions for community feedback, beta signups, and Q&A.

## Enable Discussions

1. Go to the repo **Settings** → **General**.
2. Scroll to **Features**.
3. Check **Discussions**.
4. Click **Set up discussions** and create categories if needed (e.g. **Ideas**, **Q&A**, **Beta feedback**).

## Suggested categories

- **Ideas** — Feature ideas and RFCs  
- **Q&A** — How-to and troubleshooting  
- **Beta feedback** — For beta testers (link from `docs/beta-testers.md`)  
- **Show and tell** — Workflows and demos  

This file can be removed or kept as a manual note; there is no API to enable Discussions from a script.
