# PR Title & Description Convention

Roland autogenerates draft pull requests when executor tasks complete the task git
workflow (`src/rco/task-git-workflow.ts`). Titles and bodies follow this convention.

## PR titles

**Format:** `type(scope): short imperative description`

| Part | Rule |
|------|------|
| `type` | Inferred from task wording: `feat` (default), `fix`, `refactor`, `docs`, `test`, `chore` |
| `scope` | Inferred area: `dashboard`, `git`, `loop`, `pm`, `mcp`, `api`, `auth`, or `roland` |
| `description` | Task title with mission noise stripped; lowercase imperative; max 72 chars total |

**Good examples**

```
feat(git): implement clean pr title and description convention
fix(dashboard): restore mobile responsiveness on iphone
refactor(loop): simplify critique escalation logic
```

**Avoid (legacy — auto-migrated when possible)**

```
Task task-1: [Mission: PR-cleanup] Team Goal: Improve PR Title…
Roland: Add rate limiting to the Express API
```

## PR descriptions

Autogenerated bodies include:

1. **Summary** — one paragraph from the task description
2. **Key changes** — bullet list from task scope
3. **Testing** — Roland workflow notes (Vanguard may append specifics)
4. **Related** — task id, agent, mission goal, run id, dashboard URL

Implementation: `buildPrDescription()` in `src/rco/pr-format.ts`.

## Commit messages

Per-task commits use the same conventional title as the PR, plus:

```
Closes task task-1

Generated by Roland
```

## Retroactive cleanup

Rename recent open PRs that still use legacy titles:

```bash
# Preview changes (default — dry-run)
roland pr-cleanup

# Apply via GitHub CLI
roland pr-cleanup --apply

# Scan fewer PRs
roland pr-cleanup --limit 10 --apply
```

Requires `gh` authenticated against the repo. PRs already matching the convention are skipped.

## Manual migration

When batch rename is not available, edit the title in GitHub to:

`type(scope): <stripMissionNoise(original title)>`

Use `migrateLegacyPrTitle()` from `src/rco/pr-format.ts` in a REPL or one-off script to preview the suggested title.
